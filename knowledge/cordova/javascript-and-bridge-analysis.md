# Cordova JavaScript and Bridge Analysis

Inventory `www`, `config.xml`, plugin metadata, JavaScript, and native plugin code. Map JavaScript
bridge service/action calls to plugin declarations and Android implementations. Inspect allowlists,
content origins, navigation, endpoints, and sensitive native APIs. Configuration paths and WebView
origin behavior vary by Cordova Android version, so record the version and avoid fixed assumptions.
