# Cocos Script Artifact Analysis

Identify engine generation, JavaScript versus Lua, and whether scripts are plaintext, compiled,
encrypted, or existing decompiler output. Search and read plaintext or researcher-provided output in
bounded ranges. Connect scripts to native bindings only when both sides provide evidence. Clearly
report unsupported compiled or encrypted formats; do not label arbitrary binary data as source.
