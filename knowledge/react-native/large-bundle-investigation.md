# Large React Native Bundle Investigation

Never read a complete large bundle into model context. Inspect metadata, run exact searches, index
chunks or Metro modules, inspect selected contexts, and persist large result sets under
`tool-output`. Reindex whenever size or modification metadata changes. Use multiple independent
indicators before assigning behavioral confidence to minified code.
